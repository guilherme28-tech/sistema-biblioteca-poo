# Sistema de Biblioteca em Python

Projeto desenvolvido para praticar Programação Orientada a Objetos (POO) em Python.

## Descrição

Sistema simples para gerir uma biblioteca via terminal.  
Permite cadastrar livros e usuários, realizar empréstimos e devoluções de livros, e listar registros.

## Funcionalidades

- Cadastro de livros
- Cadastro de usuários
- Empréstimo de livros
- Devolução de livros
- Listagem de livros
- Listagem de usuários

## Estrutura do Projeto

sistema-biblioteca/

│   ├── livro.py

│
└── main.py

## Conceitos aplicados

- Classes e objetos
- Encapsulamento

- Controle de estado
- Estrutura modular

## Requisitos

- Python 3.10+

## Como executar

No terminal, dentro da pasta do projeto:

python main.py
